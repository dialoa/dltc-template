# dltc-template

Pandoc templates, Lua filters, and settings used to compile articles for the journal [Dialectica](https://dialectica.philosophie.ch/).

## Versions

- `v1.0` — initial template
- `v1.1` — added HTML output, new filters
- `v1.2` — production (Pandoc 3.5)
- `dev` branch — experimental (Pandoc 3.6+)
